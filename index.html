พร้อมขออนุญาตใช้กล้อง

แตะปุ่มสีม่วงด้านล่าง จากนั้นกด “อนุญาต” เมื่อโทรศัพท์ถามสิทธิ์ กล้องหลังจะถูกเลือกก่อนโดยอัตโนมัติ

รอเริ่มกล้อง
⌁ พร้อมสแกน ยังไม่มีรายการ วาง QR Code หรือบาร์โค้ด Code 128 ให้อยู่กลางกรอบ
ข้อมูลส่วนตัวปลอดภัย: หน้านี้ไม่เก็บฐานข้อมูลนักเรียน ไม่เก็บ PIN และไม่เก็บ token ของระบบ ส่งกลับเฉพาะรหัสที่กล้องอ่านได้ผ่านหน้าต่างที่เปิดมาจากระบบหลักเท่านั้น
